📊 SRM Algorithm Comparison Matrix

Quick-reference guide for selecting computational statistics and Monte Carlo algorithms for DDS 6106.

Algorithm Track Primary Use Case Key Advantage / Strength Main Limitation / Trap
Inverse-Transform Track 1 Sampling 1D continuous random variates 100% efficient; no rejected draws Requires analytical inverse CDF F^-1(u)
Accept-Reject Track 2 Sampling complex non-standard densities Works even when F^-1(u) is unknown High constant c causes many rejected draws
Control Variates Track 2 Variance reduction in Monte Carlo integration Reduces variance by (1 - rho^2) for correlated controls Requires a control h(X) with known mean mu_h
Antithetic Variates Track 2 Variance reduction using negative correlation Zero extra random numbers required Only works for monotonic integrand functions
Importance Sampling Track 2 Estimating rare event probabilities & tail risks Dramatically reduces variance in extreme tails Unbounded weight ratios f(x)/g(x) cause variance explosion
Bootstrap Track 1 Non-parametric confidence intervals & SEs No distributional assumptions needed Requires i.i.d. random sampling
Metropolis-Hastings Track 3 High-dimensional Bayesian posterior sampling Only requires unnormalized log target density Autocorrelated samples; requires burn-in tuning
Gibbs Sampling Track 3 Multi-parameter MCMC sampling 100% acceptance rate; no proposal tuning needed Requires known full conditional distributions
Coupling From The Past (CFTP) Track 3 Exact un-biased MCMC sampling Completely eliminates burn-in bias Requires monotonic state space coupling
Robbins-Monro Track 3 Stochastic root-finding & optimization Finds roots under random measurement noise Step size decay sequence a_n = c/n must be tuned
